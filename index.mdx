Baritone is the pathfinding system used in [Impact](https://impactclient.net/) and many other Minecraft utility clients. It calculates paths in real time using a heavily optimized A\* algorithm, handles block breaking and placing, and exposes a clean Java API so other mods can control it programmatically.

## Integrate Baritone into your mod

Baritone ships with a public Java API so utility clients and mods can control it directly.

<Steps>
  <Step title="Add the API jar as a dependency">
    Grab `baritone-api-*-VERSION.jar` from the [releases page](https://github.com/cabaletta/baritone/releases) and add it to your Gradle project.
  </Step>
  <Step title="Get the Baritone instance">
    Use `BaritoneAPI.getProvider().getPrimaryBaritone()` to access the active instance.
  </Step>
  <Step title="Set a goal and path">
    Create a `Goal` (e.g. `GoalXZ`, `GoalBlock`) and pass it to the `ICustomGoalProcess`.
  </Step>
</Steps>
