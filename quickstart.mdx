---
title: "Quick Start"
description: "Get Baritone running in under a minute."
---

Once Baritone is installed, you control it through in-game chat using the `#` prefix. Type a command, press Enter, and Baritone starts moving.

<Tip>
  Always use the `#` prefix. Without it, if you mistype a command (e.g. `gola` instead of `goto`), it goes into public chat. The prefix keeps your commands local.
</Tip>

<Steps>
  <Step title="Open Minecraft and join a world">
    Launch Minecraft with your Forge, NeoForge, or Fabric profile and load into any world — singleplayer or multiplayer.

    To confirm Baritone loaded correctly, open chat and type:

    ```
    #version
    ```

    Baritone replies with its version number in chat.
  </Step>

  <Step title="Navigate to coordinates">
    Tell Baritone to walk to X=1000, Z=500:

    ```
    #goto 1000 64 500
    ```

    Baritone calculates a path and starts moving immediately. Watch the green path overlay appear in the world.

    You can also navigate to just an X/Z without specifying Y:

    ```
    #goto 1000 500
    ```
  </Step>

  <Step title="Mine a resource">
    To mine diamond ore, type:

    ```
    #mine diamond_ore
    ```

    Baritone searches its chunk cache for nearby diamond ore and paths to it. To mine a specific quantity, add the count before the block name:

    ```
    #mine 64 diamond_ore
    ```

    Baritone stops automatically once it has collected 64 diamond ore.
  </Step>

  <Step title="Stop Baritone">
    To cancel whatever Baritone is doing:

    ```
    #stop
    ```

    `#cancel` works the same way. Use `#forcecancel` if Baritone does not stop immediately.
  </Step>
</Steps>

## Common first commands

| Command | What it does |
|---------|--------------|
| `#goto x z` | Navigate to X, Z coordinates at any Y |
| `#goto x y z` | Navigate to a specific block position |
| `#mine <block_type>` | Mine blocks of a given type indefinitely |
| `#mine 64 diamond_ore` | Mine exactly 64 diamond ore, then stop |
| `#stop` / `#cancel` | Stop all current activity |
| `#help` | List all commands — clickable, with tab completion |

<Note>
  **Impact client users:** In Impact, use `.b` as the prefix instead of `#`. For example, `.b goto 1000 500` instead of `#goto 1000 500`. Direct chat control (no prefix) is disabled by default in Impact.
</Note>
